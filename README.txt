Financial expenses tracking program

This program helps you track your expenses. Add, save and view your financial expenses. Reports feature coming soon.
It started as a simple tracker for the final project for Harvard's CS50x Course, but by adding new features I realised I want to expand it more
and use it to practice my automation testing skills.
