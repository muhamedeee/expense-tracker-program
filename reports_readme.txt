EXPENSES TRACKER PROJECT - EXPENSES TRACKER WITH REPORTS

This latest release has "REPORTS" feature integrated, that shows us how much total expenses we made
and expenses by type.

HOW TO USE:

IN MAIN MENU, SELECT OPTION '3. REPORTS'
CHOOSE AN OPTION '1. TOTAL EXPENSES REPORT' OR '2. EXPENSES BY TYPE REPORT'
IF YOU WANT TO GO BACK TO MAIN MENU, CHOOSE OPTION '3. MAIN MENU'.

ALSO, REMOVED HEROBRINE.
